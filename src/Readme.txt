Purpose of application: Appointment scheduling application. Can be used to create appointments and track customers and their appointments.
IDE:
IntelliJ IDEA 2020.3.1 (Ultimate Edition)
JDK 11
JavaFX SDK 11.0.2
MYSQL JDBC Connector 8.0.22

To run the program:
Click the run button and log in with the username: test password: test
The Javadoc information can be found in the Javadoc directory in the root folder. Open it and click on index.html to see
the documentation.

Additional Report Description
View both user's appointment schedule. The logged in user can see all appointments for both users sorted by month and time,
and can follow a list of their appointments.

Lambdas:
Use of lambdas can be found in the GeneralErrorMessage interface in the main menu, ShowTimeZoneID interface on main menu
and ForEach functions in addToList Method in MonthTypeBrief controller and UserScheduleController. Javadoc comments can be found
in their respective locations.
